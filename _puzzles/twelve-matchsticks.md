Twelve matchsticks can be laid on the table to produce a variety of shapes. For example:

~~~
                               __
 __ __          __ __ __      |  |
|     |__      |        |     |  |
|__    __|     |        |     |  |
   |__|        |__ __ __|     |  |
                              |__|
~~~
Some shapes have the same area and some have a different area. 

**Arrange 12 matchsticks to make a shape with area 4.**
